# 오류 현황

## 개요

로그 적재 과정에서 발생한 오류를 모아서 보여주는 페이지입니다.
5초마다 자동 갱신되어 실시간으로 오류를 모니터링할 수 있습니다.

## 전제조건

- Fastify 서버가 실행 중이어야 합니다
- Oracle DB가 연결되어 있어야 합니다

## 화면 구성

### 오류 테이블

각 오류 항목에는 다음 정보가 표시됩니다:

| 컬럼 | 설명 |
|------|------|
| **시간** | 오류 발생 일시 |
| **테이블** | 적재 대상이었던 Oracle 테이블명 |
| **장비** | 오류가 발생한 설비 ID |
| **메시지** | 상세 오류 내용 |

### 전체 삭제

1. 화면 상단의 **전체 삭제** 버튼을 클릭합니다
2. 확인 팝업에서 **확인**을 클릭합니다
3. 모든 오류 로그가 삭제되고 삭제 건수가 표시됩니다

> 삭제된 오류는 복구할 수 없습니다. 필요 시 오류 내용을 먼저 기록해 두세요.

## 자주 발생하는 오류 유형

| 오류 메시지 | 원인 | 해결 방법 |
|-------------|------|-----------|
| ORA-00942: table or view does not exist | 대상 테이블이 없음 | 타겟 매핑에서 테이블 확인 |
| ORA-01400: cannot insert NULL | 필수 컬럼에 값 없음 | 매핑에서 SOURCE_FIELD 확인 |
| ORA-12899: value too large | 데이터 길이 초과 | Oracle 컬럼 크기 조정 |
| Connection refused | DB 연결 실패 | Oracle 서버 상태 확인 |
