# 송신기 설정

## 개요

설비 PC에 배포할 Vector Agent TOML과 Fluent Bit 설정을 생성·검증·관리합니다.

## 기본 절차

1. 왼쪽에서 송신기 종류와 설비를 선택하거나 새 설비를 추가합니다.
2. 설비 유형, 로그 유형, 라인 코드와 설비 ID를 입력합니다.
3. 중앙 서버 IP와 포트, 읽기 시작 위치와 타임아웃을 설정합니다.
4. 로그 포함 경로와 제외 경로를 줄 단위로 입력합니다.
5. 필요하면 전체 파일 묶기, 재귀 검색, fingerprint, multiline과 재전송 폴더를 조정합니다.
6. disk/memory buffer, full 정책, 압축과 acknowledgement를 확인합니다.
7. 저장 시 설정 검증 결과를 확인한 뒤 다운로드합니다.

Fluent Bit 구성은 tail 입력, multiline, storage와 forward 전송 옵션을 별도로 제공합니다.

> 설비 삭제는 서버의 설정 파일을 제거합니다. 배포 중인 파일이 필요하면 먼저 다운로드하세요.
