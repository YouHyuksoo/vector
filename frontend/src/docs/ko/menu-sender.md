# 송신기 설정

## 개요

각 설비 PC에 설치할 Agent의 TOML 설정을 생성하고 관리하는 페이지입니다.
설비별로 로그 경로, 서버 연결 정보, 메타데이터를 설정합니다.

## 전제조건

- Fastify 서버가 실행 중이어야 합니다

## 화면 구성

### 좌측: 설비 목록

등록된 모든 설비가 목록으로 표시됩니다.

#### 설비 추가

1. **추가** 버튼을 클릭합니다
2. 설비 이름을 입력합니다 (영문 대문자, 숫자, 하이픈만 허용)
   - 예: `AOI-LINE01`, `SP-001`, `REFLOW-A`
3. 확인을 클릭하면 기본 TOML 템플릿이 생성됩니다

#### 설비 삭제

1. 삭제할 설비를 선택합니다
2. **삭제** 버튼을 클릭합니다
3. 확인 팝업에서 **확인**을 클릭합니다

> 삭제하면 설정 파일이 영구 삭제됩니다. 복구할 수 없으니 주의하세요.

### 우측: Agent 설정 편집

설비를 선택하면 해당 설비의 TOML 설정을 편집할 수 있습니다.

#### 설비 정보

| 항목 | 설명 | 예시 |
|------|------|------|
| 설비 유형 | 설비 종류 | SP, SPI, AOI, REFLOW 등 |
| 로그 유형 | 로그 분류 | INSPECTION, ALARM 등 |
| 라인 코드 | 생산 라인 식별자 | LINE-01, L1 |
| 설비 코드 | 설비 고유 ID | AOI-001, SP-A01 |

#### 연결 설정

| 항목 | 설명 | 기본값 |
|------|------|--------|
| 서버 IP | Aggregator 서버 주소 | `192.168.1.10` |
| 포트 | Aggregator 수신 포트 | `9000` |
| 읽기 모드 | `end`(새 줄만) / `beginning`(전체) | `end` |
| 파일 만료 | 오래된 파일 무시 기준 (초) | `86400` (24시간) |
| 타임아웃 | 읽기 타임아웃 (ms) | `1000` |
| 버퍼 | 전송 버퍼 크기 (MB) | `256` |

#### 로그 경로

감시할 로그 파일 경로를 줄 단위로 입력합니다.

- glob 패턴 지원: `C:/logs/*.csv`, `D:/data/**/*.log`
- Windows 경로와 Unix 경로 모두 사용 가능
- 여러 경로는 줄바꿈으로 구분합니다

#### TOML 직접 편집

**TOML 직접 편집** 토글을 열면 설정 파일의 전체 TOML 내용을 직접 편집할 수 있습니다.

### 설정 저장

**저장** 버튼을 클릭하면 서버에 TOML 파일이 저장됩니다.

### 설정 다운로드

**다운로드** 버튼을 클릭하면 TOML 파일을 로컬 PC에 다운로드할 수 있습니다.
다운로드한 파일을 설비 PC에 복사하여 사용합니다.
