# 원본 로그 파일

## 개요

서버의 원본 로그 저장소를 폴더별로 탐색하고 파일을 조회·검색·다운로드·삭제하거나 수동 투입하는 화면입니다.

## 사용 방법

1. 왼쪽 탐색기에서 설비 유형, 설비 ID, 날짜 폴더를 차례로 엽니다.
2. 파일을 선택하면 오른쪽에서 내용과 행 수를 확인할 수 있습니다.
3. 검색어를 입력해 선택 파일 안의 내용을 찾습니다.
4. 체크한 파일은 일괄 다운로드 또는 삭제할 수 있습니다.

## 수동 투입

파일을 선택한 뒤 **수동 투입**에서 설비 유형과 설비 ID를 지정합니다. 선택한 유형에 등록된 VRL과 타겟 매핑이 있어야 실행할 수 있습니다.

> 삭제는 원본 파일을 실제로 제거합니다. 재처리가 필요하면 먼저 다운로드해 보관하세요.
