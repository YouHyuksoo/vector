# 로그 조회

## 개요

Oracle DB에 적재된 로그 데이터를 테이블별로 조회할 수 있는 페이지입니다.
수집된 데이터가 정상적으로 저장되었는지 확인하는 데 사용합니다.

## 전제조건

- Oracle DB가 연결되어 있어야 합니다
- 최소 1개 이상의 로그 테이블이 Oracle에 존재해야 합니다
- **타겟 매핑**에서 테이블 매핑이 설정되어 있어야 데이터가 적재됩니다

## 사용 방법

### 1단계: 테이블 선택

화면 상단의 드롭다운에서 조회할 테이블을 선택합니다.

- Oracle DB에 존재하는 모든 로그 테이블이 목록에 표시됩니다
- `LOG_` 접두사가 붙은 테이블들이 자동 생성된 로그 테이블입니다

### 2단계: 조회 건수 설정

- 기본값: **50건**
- 최대: **500건**
- 숫자를 직접 입력하여 변경할 수 있습니다

### 3단계: 데이터 확인

테이블을 선택하면 즉시 최신 데이터가 표시됩니다.

- 모든 컬럼이 수평 스크롤 가능한 테이블로 표시됩니다
- 긴 텍스트 값은 자동으로 잘려서 표시됩니다
- 하단에 조회된 행 개수가 표시됩니다

### 새로고침

**새로고침** 버튼을 클릭하면 최신 데이터를 다시 불러옵니다.

## 활용 팁

- 수집기 설정 변경 후 데이터가 정상 적재되는지 확인할 때 유용합니다
- 오류 현황에서 실패한 건이 있을 경우, 해당 테이블을 조회하여 마지막으로 성공한 데이터를 확인할 수 있습니다
