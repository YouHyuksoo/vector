# 파일 업로드

## 개요

설비 ID를 지정해 운영 파일을 서버에 업로드하고 업로드 이력을 관리하는 화면입니다.

## 업로드

1. 설비 ID를 입력합니다.
2. 파일을 끌어 놓거나 파일 선택 창에서 하나 이상 고릅니다.
3. 선택 목록과 크기를 확인한 뒤 업로드합니다.

## 이력 관리

- 전체 또는 설비 ID별로 업로드 파일을 필터링합니다.
- 개별 파일을 다운로드하거나 삭제할 수 있습니다.
- 필터된 설비의 파일 또는 전체 파일을 일괄 삭제할 수 있습니다.

> 일괄 삭제는 복구할 수 없으므로 대상 설비와 파일을 먼저 확인하세요.
