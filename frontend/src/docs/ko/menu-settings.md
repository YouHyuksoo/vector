# 시스템 설정

## 관리 항목

- **서버**: bind host, API port, 실행 환경과 현재 프로세스 정보
- **Oracle**: connect string, 사용자, 비밀번호, pool 최소/최대
- **저장소**: 원본 로그 루트 `RAW_LOG_BASE_PATH`
- **하트비트**: 온라인 판정 TTL

**편집**에서 값을 변경하고 저장합니다. 서버/Oracle 항목은 재시작이 필요할 수 있으며, 비밀번호는 마스킹된 값 그대로 저장하면 변경하지 않습니다.

Oracle 카드의 **연결 테스트**는 실제 `SELECT 1 FROM DUAL` 요청과 응답 시간을 확인합니다.

## AI 모델

VRL 코드 생성에 사용할 제공사, API 키와 모델을 설정하고 연결을 테스트합니다. AI가 만든 코드는 초안이므로 **VRL & 매핑**에서 시뮬레이션을 통과한 뒤 적용하세요.

> 현재 런타임은 Redis/큐 설정을 사용하지 않습니다.
