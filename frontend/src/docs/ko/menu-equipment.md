# 장비 대시보드

## 개요

등록된 설비의 Agent 하트비트와 수집 상태를 장비별로 확인하는 화면입니다.

## 상태 확인

- **온라인**: 최근 하트비트가 정상적으로 도착한 설비
- **오프라인**: 하트비트 제한 시간을 지난 설비
- **수집 제외**: 원본 파일과 수신 이력은 남기되 Oracle 적재를 건너뛰는 설비

장비 카드를 선택하면 설비 ID, 유형, 라인, 마지막 하트비트와 처리 통계를 확인할 수 있습니다.

## 운영 작업

- **수집 제외/재개**: 선택 설비의 DB 적재 여부를 전환합니다.
- **등록 삭제**: 더 이상 사용하지 않는 설비를 레지스트리에서 제거합니다. Agent가 다시 하트비트를 보내면 재등록될 수 있습니다.

> 수집 제외는 Agent나 파일 수신을 중지하는 기능이 아닙니다. DB 적재만 건너뜁니다.
