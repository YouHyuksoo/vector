# 시스템 로그

## 개요

처리 실패, 재전송, 전체 처리 이력, 실시간 서버 로그와 PM2 로그 파일을 한 화면에서 확인합니다.

## 탭

- **오류**: `ERROR` 처리 이력과 원본 데이터
- **재전송**: 실패 로그 재처리 결과
- **처리 이력**: 파일 수신, HTTP 수신, TABLE INSERT, PROCEDURE CALL 등의 단계별 기록
- **실시간**: 서버 메모리 로그를 레벨·검색어로 필터링하며 자동 새로고침 가능
- **PM2 파일**: 운영 프로세스의 디스크 로그 파일 조회

오류를 조사할 때는 발생 시각, 설비 ID, stage, 대상 테이블과 오류 메시지를 함께 확인하세요.
