# VRL & Mapping

Select an equipment type, then manage parsing and Oracle mapping in one screen.

## VRL tab

Enter a sample log, write or generate VRL, run the simulator, and apply only validated code. Reload the Vector Aggregator when prompted.

## Mapping tab

Choose TABLE or PROCEDURE, map `data.*` fields to Oracle columns or parameters, then save. A missing target can be created from the screen.

> Always run the simulator before applying VRL to the live aggregator.
