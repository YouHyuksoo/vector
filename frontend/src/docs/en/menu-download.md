# Agent Download

Download Vector x64 (`vector.zip`), Vector x86 (`vector-x86.zip`), Agent Manager x64/x86, Fluent Bit, and the `.toml` or `.conf` files managed in Sender Configuration.

Recommended deployment: run Agent Manager, install Vector to `C:\vector`, place the equipment configuration there, verify paths and server address, start Vector, register the service if needed, and confirm online status centrally.
