# System Settings

Manage server binding, Oracle connection and pool, `RAW_LOG_BASE_PATH`, and heartbeat TTL. Edit and save values; server or Oracle changes may require restart. Masked passwords remain unchanged unless replaced.

The Oracle connection test executes a live query and reports latency.

Configure and test the AI provider used for VRL generation below the system cards. Simulate generated VRL before applying it. Redis and queue settings are not used by the current runtime.
