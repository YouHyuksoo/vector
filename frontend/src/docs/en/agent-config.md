# Agent Configuration

Vector Agent or Fluent Bit watches equipment log files and forwards them to the central Aggregator.

Set equipment type, log type, line code, equipment ID, include/exclude globs, read position, and the central address. The Sender screen also exposes fingerprint, multiline, recursive scan, resend folder, disk buffer, compression, and acknowledgement options.

Validate before saving and downloading. Do not reset `data_dir` or fingerprint state casually, because doing so may cause files to be read again.
