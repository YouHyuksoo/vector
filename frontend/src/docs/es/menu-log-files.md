# Archivos de Log Raw

Explore el repositorio por tipo de equipo, ID y fecha. Seleccione un archivo para ver y buscar su contenido.

Los archivos marcados pueden descargarse o eliminarse en grupo. **Ingesta manual** procesa el archivo con el VRL y mapeo registrados para el tipo elegido.

> La eliminación borra el archivo original. Descargue una copia si puede necesitarlo.
