# Descarga de Agent

Descargue Vector x64 (`vector.zip`), Vector x86 (`vector-x86.zip`), Agent Manager x64/x86, Fluent Bit y configuraciones `.toml` o `.conf` gestionadas en Emisor.

Flujo recomendado: ejecutar Agent Manager, instalar Vector en `C:\vector`, colocar la configuración, revisar rutas y servidor, iniciar Vector, registrar el servicio si hace falta y confirmar el estado central.
