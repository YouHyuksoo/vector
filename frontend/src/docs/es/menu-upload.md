# Carga de Archivos

Indique el ID del equipo, seleccione o arrastre uno o más archivos, revise la lista y súbalos al servidor.

El historial se puede filtrar por equipo. Es posible descargar o eliminar archivos individuales, los del filtro actual o todos.

> La eliminación masiva no se puede deshacer. Verifique el filtro antes de continuar.
