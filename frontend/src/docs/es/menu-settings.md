# Configuración del Sistema

Gestione binding del servidor, conexión y pool Oracle, `RAW_LOG_BASE_PATH` y TTL heartbeat. Los cambios de servidor u Oracle pueden requerir reinicio. La contraseña enmascarada no cambia si no se reemplaza.

La prueba Oracle ejecuta una consulta real y muestra latencia.

Configure y pruebe el proveedor AI para generar VRL. Simule el resultado antes de aplicarlo. Redis y cola no se usan en el runtime actual.
