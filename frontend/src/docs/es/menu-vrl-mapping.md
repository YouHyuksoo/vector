# VRL y Mapeo

Seleccione un tipo de equipo y gestione el parsing y el mapeo Oracle en una sola pantalla.

En la pestaña **VRL**, ingrese un log de muestra, escriba o genere el código, ejecute la simulación y aplique solamente código validado. Recargue Vector Aggregator cuando se solicite.

En **Mapeo**, elija TABLE o PROCEDURE, conecte campos `data.*` con columnas o parámetros Oracle y guarde.
