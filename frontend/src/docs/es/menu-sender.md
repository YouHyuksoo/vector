# Configuración del Emisor

Cree y gestione TOML de Vector Agent y configuración Fluent Bit.

Seleccione o agregue equipo, ingrese metadatos y conexión, configure include/exclude, detección, multiline, reenvío y buffer; después valide, guarde y descargue.

Fluent Bit presenta por separado tail, multiline, storage y forward. Descargue una configuración necesaria antes de eliminarla del servidor.
