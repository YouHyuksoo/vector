# Panel de Equipos

Supervise el heartbeat del Agent y el estado de recolección de cada equipo registrado. Seleccione una tarjeta para ver metadatos, último heartbeat y estadísticas.

- **En línea**: el heartbeat llegó dentro del TTL configurado.
- **Fuera de línea**: el TTL expiró.
- **Excluido**: se conservan archivos raw e historial, pero se omiten los INSERT en Oracle.

Puede cambiar la exclusión o eliminar un registro obsoleto. El equipo puede registrarse otra vez con un nuevo heartbeat.
